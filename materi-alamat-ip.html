Alamat IP (versi 4)

1. Kelas Alamat IP

Ada lima kelas utama alamat IP, yaitu Kelas A, B, C, D, dan E. Namun, hanya Kelas A, B, dan C yang umum digunakan untuk pengalamatan perangkat dalam jaringan. Kelas D digunakan untuk IP multicasting dan kelas E dicadangkan untuk eksperimen.
Penentuan alamat IP adalah dengan melihat nilai dari oktet pertama sebuah alamat IP.

Kelas A: Rentang IP dari 0 s.d. 127 (IP 0 dan 127 dicadangkan)

Kelas B: Rentang IP dari 128 s.d. 191

Kelas C: Rentang IP dari 192 s.d. 223

Kelas D: Rentang IP dari 224 s.d. 239

Kelas E: Rentang IP dari 240 s.d. 255

2. Prefix length dan Subnet Mask

Prefix length adalah bagian dari alamat IP yang menunjukkan jaringan tempat suatu perangkat berada. Prefix menunjukkan banyaknya binary 1 secara berurutan dimulai dari bit pertama. Prefix digunakan untuk menentukan subnet mask dari alamat IP.

Subnet mask digunakan untuk membedakan network ID dengan host ID, menunjukkan letak suatu host, apakah berada di jaringan lokal atau jaringan luar.

Bit-bit subnet mask yang didefinisikan, adalah sebagai berikut:
Semua bit yang digunakan oleh Network Id memiliki nilai 1.
Semua bit yang digunakan oleh Host Id memiliki nilai 0.

Subnet mask bawaan dari kelas IP

Kelas IP Subnet Mask Biner Subnet Mask
A 11111111.00000000.00000000.00000000 255.0.0.0
B 11111111.11111111.00000000.00000000 255.255.0.0
C 11111111.11111111.11111111.00000000 255.255.255.0

3. Alamat Network dan Host

Setiap alamat IP terdiri dari 2 bagian, Alamat Network dan Alamat Host

Alamat Network atau Network ID merupakan bagian dari alamat IP yang mengidentifikasi jaringan tertentu di mana sebuah perangkat berada. Semua perangkat jaringan harus mempunyai alamat network yang sama untuk bisa saling terhubung.

Alamat Host atau Host ID merupakan bagian dari alamat IP yang mengidentifikasi perangkat spesifik secara individu. Dalam jaringan yang sama, tidak boleh ada 2 atau lebih alamat host yang sama.

Kelas A

Net ID . HostID . HostID . HostID

Kelas B

Net ID . Net ID . HostID . HostID

Kelas C

Net ID . Net ID . Net ID . HostID

Penulisan Alamat IP

Penulisan IP address umumnya adalah dengan 192.168.1.2.
Namun adakalanya ditulis dengan 192.168.1.2/24.
Artinya adalah IP address 192.168.1.2 dengan subnet mask 255.255.255.
/24 berarti bahwa 24 bit pertama subnet mask merupakan binari 1.
Atau dengan kata lain, subnet masknya adalah: 11111111.11111111.11111111.00000000. Yang dikonversi menjadi bilangan desimal menjadi 255.255.255.0.
