Pemecahan Alamat IP (Subnetting)

Subnetting adalah proses membagi satu jaringan besar (network) menjadi beberapa jaringan kecil (subnetwork atau subnet). Salah satu tujuannya adalah efisiensi alamat IP, agar alamat IP yang tidak digunakan tidak terbuang percuma.

Dalam kelompok alamat IP ada 2 buah IP yang mempunyai sifat khusus yaitu:
IP Network: Alamat IP yang merupakan identitas dari kelompok IP. Merupakan Alamat IP terkecil dari sebuah jaringan.
IP Broadcast: Alamat IP yang digunakan untuk memanggil semua IP dalam suatu kelompok. Merupakan Alamat IP terbesar dari sebuah jaringan.

B. Menentukan Subnetmask, IP Network, IP Broadcast dan IP Host (Kelas C)

1. Menghitung Blok Subnet dan Subnetmask

a. Cara Pertama
Menghitung Subnetmask
Ubah prefix menjadi subnetmask biner lalu konversikan menjadi desimal.

Contoh: 192.168.10.150/26
Subnetmask biner = 11111111.11111111.11111111.11000000
                   26 bit pertama merupakan bilangan 1, bit tersisa merupakan bilangan 0
Dikonversi menjadi bilangan desimal
Subnetmask       = 255.255.255.192

Menghitung Blok Subnet

Blok Subnet = 256 - Oktet ke-4 Subnetmask

Contoh: 192.168.10.150/26
Blok subnet = 256 - Oktet ke-4 Subnetmask
            = 256 - 192
Blok Subnet = 64

b. Cara Kedua
Menghitung Blok Subnet

Blok Subnet = 2 (32 - prefix)

Contoh: 192.168.10.150/26
Blok subnet = 2(32 - prefix)
            = 2(32 - 26)
            = 26
Blok Subnet = 64

Menghitung Subnetmask

Subnetmask = 255.255.255.(256-Blok Subnet)

Contoh: 192.168.10.150/26
Subnetmask  = 255.255.255.(256-Blok Subnet)
            = 255.255.255.(256-64)
            = 255.255.255.192

2. Menghitung IP Network

Oktet ke-1 sampai oktet ke-3 tidak diubah, oktet ke-4 mengikuti perhitungan berikut.

Nilai oktet ke-4 / Blok subnet

Angka sebelum koma x Blok subnet

Contoh: 192.168.10.150/26
Nilai Oktet ke-4 / Blok Subnet   = 150 / 64
                                 = 2,343
Angka Sebelum Koma x Blok Subnet = 2 x 64
                                 = 128

IP Network = 192.168.10.128

3. Menghitung IP Broadcast

Nilai oktet ke-4 IP Network + Blok subnet - 1

Contoh: 192.168.10.150/26
Nilai Oktet ke-4 IP Network + Blok Subnet - 1  = 128 + 64 - 1
                                               = 191

IP Broadcast = 192.168.10.191

4. Menghitung IP Host

IP Host adalah alamat IP yang bisa digunakan untuk konfigurasi perangkat jaringan.
Merupakan Alamat IP yang berada diantara IP Network dengan IP Broadcast

Contoh: 192.168.10.150/26
IP Network              IP Broadcast
192.168.10.128          192.168.10.191
           +1                      -1
192.168.10.129  sampai  192.168.10.190

IP Host = 192.168.10.129 - 192.168.10.190
Hasil akhir perhitungan alamat IP diatas
Alamat IP    = 192.168.10.150/26
Subnetmask   = 255.255.255.192
Blok Subnet  = 64
IP Network   = 192.168.10.128
IP Broadcast = 192.168.10.191
IP Host      = 192.168.10.129 - 192.168.10.190
