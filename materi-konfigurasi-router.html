Konfigurasi Dasar Router

1. Reset Konfigurasi MikroTik

Menghapus semua konfigurasi Router MikroTik dan mengembalikan ke pengaturan awal.

a. System → Reset Configuration.

b. Beri tanda ✔ pada No Default Configuration.

c. Klik tombol Reset Configuration → Yes.

d. Tunggu beberapa saat dan koneksi dengan Router akan terputus. Kemudian ulangi proses menhubungkan Router dari awal.

2. Konfigurasi Hostname

Merubah nama Router MikroTik.

a. System → Identity.

b. Isikan Nama Router pada kotak Identity.

c. Klik tombol Apply → OK.

3. Konfigurasi Alamat IP

a. IP → Addressess.

b. Klik tombol + untuk menambah alamat IP.

c. Isikan sesuai ketentuan berikut:

IP Address: [Alamat IP]
Network: [Alamat Network]
Interface: [Port yang dipakai]

d. Klik tombol Apply → OK.


Contoh Alamat IP:
Address	 : 192.168.10.1/24 
Network	 : 192.168.10.0
Interface: ether1

4. Konfigurasi IP DNS

a. IP → DNS.

b. Isikan IP DNS Primer di kotak Servers.

c. Gunakan tombol segitiga dibagian kanan untuk menambah dan menghapus IP DNS.

digunakan untuk menambah IP DNS bisa menggunakan tombol

digunakan untuk menghapus IP DNS bisa menggunakan tombol

d. Beri tanda ✔ pada opsi Allow Remote Request.

bagian ini digunakan untuk menyimpan permintaan DNS dan membuat Router MikroTik sebagai DNS Server.

e. Klik tombol Apply → OK.


Contoh IP DNS yang bisa digunakan:
Google DNS     : 8.8.8.8
                 8.8.4.4
Cloudflare DNS : 1.1.1.1
                 1.0.0.1

5. Konfigurasi NAT

a. IP → Firewall → NAT.

b. Klik tombol + untuk menambah NAT.

c. Atur sesuai ketentuan berikut:

Tab General
- Chain: srcnat
- Out. Interface: [port yang digunakan]
Tab Action
- Action: masquerade

e. Klik tombol Apply → OK.

*) Konfigurasi ini diulang sesuai port yang digunakan

6. Konfigurasi Gateway

a. IP → Routes.

b. Klik tombol + untuk menambah IP Gateway.

c. Isikan IP gateway di kotak Gateway.

e. Klik tombol Apply → OK.


Contoh IP Gateway
Gateway: 192.168.10.254

7. Pengujian Koneksi Jaringan

a. New Terminal.

b. Uji koneksi jaringan menggunakan perintan ping dengan format seperti berikut:

ping [alamat tujuan]

c. Tekan Enter. 


Contoh:
Pengujian ke alamat web
    ping  [alamat web]
    ping  google.com

Pengujian ke perangkat jaringan (PC, Router, dll)
    ping  [alamat IP]
    ping  192.168.10.1
