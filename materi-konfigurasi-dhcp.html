Dynamic Host Configuration Protocol (DHCP)

Layanan yang memungkinkan perangkat dapat memberikan/meminta/menyalurkan alamat secara otomatis pada host dalam sebuah jaringan. DHCP juga mampu mendistribusikan informasi IP Address, subnetmask, IP gateway, IP DNS dan informasi lain sesuai dengan pengaturan DHCP.

A. Konfigurasi DHCP Server

1. Menambahkan IP Gateway, IP Network & IP DNS

a. IP → DHCP Server → Network.

b. Klik tombol +

c. Atur sesuai ketentuan berikut:

Address: [IP Network]
Gateway: [IP Gateway]
DNS: [IP DNS]

d. Klik tombol Apply → OK.

Contoh:
Address	 : 192.168.10.0/24 
Network	 : 192.168.10.1
DNS      : 1.1.1.1

2. Menambahkan Rentang IP

a. IP → Pool.

b. Klik tombol +

c. Atur sesuai ketentuan berikut:

Name: [Nama Rentang IP]
Addressess: [Rentang IP]

d. Klik tombol Apply → OK.

Contoh:
Name	   : rentang_dhcp 
Addressess : 192.168.10.11-192.168.10.30

3. Membuat DHCP Server

a. IP → DHCP Server → DHCP.

b. Klik tombol +

c. Atur sesuai ketentuan berikut:

Name: [Nama DHCP Server]
Interfaces: [Port yang dipakai]
Lease Time: [Lama Waktu DHCP]
Address Pool: [Nama Rentang IP]

d. Klik tombol Apply → OK.

Contoh:
Name	     : dhcp_lab_tkj 
Interfaces   : ether2
Lease Time   : 00:20:00 
Address Pool : rentang_dhcp
